== Description ==
<strong>We are no longer developing or supporting the plug-in.</strong>

Braintree Payments for WordPress utilizing Braintree's Drop-in UI, allowing a quick and easy way to securely accept one-time Credit Card and PayPal payments.

This plugin allows you to seamlessly accept one-time payments on your site using your Braintree account and is:
- Customizable to fit your brand
- Localized for 23 languages
- Streamline PCI DSS compliance

Feel free to try out the plugin through our installation of <a href="https://www.brontobytes.com/blog/braintree-payments-free-wordpress-plugin/">Braintree Payments for WordPress</a>.

== Installation ==
1. Add the plugin by uploading the zip file or by installing it from the Wordpress Plugin reposistory.
2. Activate the plugin and visit Setting > Braintree Payments to add your settings.
3. That's all!
